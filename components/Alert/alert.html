Alert An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.
Importing Use classes by adding stylesheet using link below 
            
   <link rel="stylesheet" href="https://droid.netlify.app//components/Alert/alert.css"/>
Simple Alert Simple alert component has various classes
  • primary: This is a default class which makes avatar rounded
  • error: this class makes avatar in square shape
  • success: this class makes avatar in square shape
  • warning: this class makes avatar in square shape
