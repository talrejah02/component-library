Grid A grid is used to display content in the form of multiple columns maintaining a specific ratio.
Importing Use classes by adding stylesheet using link below 
            
   <link rel="stylesheet" href="https://droid-ui.netlify.app/components/grid/Grid.css"/>
2 column Grid
  • It is a grid made up of two columns having equal width. 
            
                <div class="grid grid-two-col">
                    <div class="grid-item"><img src="../../Assets/images/avatar2.jpg" /></div>
                    <div class="grid-item"><img src="../../Assets/images/avatar2.jpg" /></div>
                </div>
