Badge Badges are small circles, positioned either at top-right or bottom-right of the parent component. Badge can be used to display numbers, online / offline status, depending on where they are used.
Importing Use classes by adding stylesheet using link below 
                    
  <link rel="stylesheet" href="https://droid-ui.netlify.app/components/badge/badge.css"/>
Profile badge Profile Badge is used to show the current status of a user. It can be used with 3 variants : online | offline | dnd 
                        
            <section class="badge">
                <img src="../../Assets/images/avatar2.jpg" class="avatar "/>
                <section class="online" ></section>
            <section/>

            <section class="badge">
                <img src="../../Assets/images/avatar2.jpg" class="avatar "/>
                <section class="offline" ></section>
            <section/>

            <section class="badge">
                 <img src="../../Assets/images/avatar2.jpg" class="avatar "/>

            <section/>
