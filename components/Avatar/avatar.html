Avatar Avatar class is used to display profile images and icons
Importing Use classes by adding stylesheet using link below 
                  
<link rel="stylesheet" href="https://droid.netlify.app//components/Avatar/avatar.css"/>
Avatar component has 1 shape class and 3 size classes
  • avatar: This is a default class which makes avatar rounded
  • square: this class makes avatar in square shape
