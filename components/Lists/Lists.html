Lists A list is a container that presents rows of data arranged in a single column.
Importing Use classes by adding stylesheet using link below 
                    
  <link rel="stylesheet" href="https://droid-ui.netlify.app/components/Lists/Lists.css"/>
Basic Lists

    Fruits

  • ORANGE
  • MANGO
  • APPLE
  • LITCHI
                    
                        <ul class="spaced-list"/>
                            <p class="list-head"/>Fruits</p/>
                            <li/>ORANGE</li>

Stacked List
  • Stacked list are generally used to display notifications where each list item is stacked on top of each other.

    NOTIFICATIONS

  • Today is the day for Apple!
  • You have 3 new invites.
  • @user followed you
  • Today is Riya's birthday
                    
                        <ul class="stacked-list">
                            <p class="list-head">NOTIFICATIONS</p>
                            <li>Today is the day for Apple!</li>
                            <li>You have 3 new invites.</li>
                            <li>@user followed you</li>
                            <li>Today is Riya's birthday</li>
                        </ul>
