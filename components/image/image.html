Image Responsive Image Components are used to dsiplay Images on websites, which are screen responsive. They expand upto the width of the container they are in.
Importing Use classes by adding stylesheet using link below 
                    
  <link rel="stylesheet" href="https://droid-ui.netlify.app/components/image/image.css"/>
Square Image
  • img: A image with default square shape
                    
  <img src="../../Assets/images/avatar2.jpg" class="img" />
Round Image
  • img round: A image with round shape 
                    
   <img src="../../Assets/images/avatar2.jpg" class="img round"/>
Responsive Image
  • img responsive:Full Width Image is the one that expands to cover the container it is present in.
Note : Resize the browser window, to see the responsive property. 
                    
   <img src="../../Assets/images/avatar2.jpg" class="img round responsive"/>
