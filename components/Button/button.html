Button Buttons are the soul of an interactive Web Application. Buttons allow users to take actions, and make choices, with a single tap.
Importing Use classes by adding stylesheet using link below 
                    
  <link rel="stylesheet" href="https://droid-ui.netlify.app/components/button/button.css"/>
Button component has 6 classes
  • Btn: A button default with default color
  • Btn-primary: A button with primary color
  • Btn-secondary: A secondary button with secondary color
                    
    <button class="btn ">default</button>
    <button class="btn btn-primary">primary</button>
    <button class="btn btn-secondary">secondary</button>
Button Link
  • Btn-link: A button default with link
                        
    <button class="btn btn-link">
        <a href="link">
        Link
        </a>
    </button>
Icon And floating button
  • Btn-icon: A button with icon
  • Btn-float: A button for floating action(you can see the demo at bottom right of the page)
                        
    <button class="btn btn-icon">
            <img src="../../Assets/svg/print-btn.svg"/>
            print 
    </button>

    <button class="btn btn-float">
            <img src="../../Assets/svg/plus-btn.svg"/> 
    </button>
