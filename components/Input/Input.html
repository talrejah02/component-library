Input Inputs are necessary for user interaction with the website. Inputs are used to take information from the user, and then save the data to the server, accordingly.
Importing Use classes by adding stylesheet using link below 
                    
  <link rel="stylesheet" href="https://droid.netlify.app//components/Input/input.css"/>
Simple input
  • input: A input default with default color
  • input-success: A input with success state
  • input-warning: A input with warning state
  • input-danger: A input with danger state
                    
  <input class="input" type="text" placeholder="primary"/>
  <input class="input input-success" type="text" placeholder="success"/>
  <input class="input input-warning" type="text" placeholder="warning"/>
  <input class="input input-danger" type="text" placeholder="danger"/>
Inline validation input
  • inline-success: for success message
  • inline-warning: Used for warning response
  • inline-danger: Used for error response
