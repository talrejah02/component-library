Modal A modal is used to show information that requires the user to take some action before moving forward.
Importing Use classes and javascript by using link below 
            
<link rel="stylesheet" href="https://droid-ui.netlify.app/components/modal/modal.css"/>
<script src="https://droid-ui.netlify.app/components/modal/modal.js" type="text/javscript"></script>
Basic Modal A basic modal has some text and 2 action buttons. Click on open modal button below to see demo.
